Proyecto 2
Reconocimiento de billetes con Haar Cascade y Python

Es un programa que usa el aprendizaje mediante haar cascade , el cual consiste en que a partir de fotos positivas y negativas puede identificar distintos objetos , para la identificación de cada billete se tomaron alrededor de 100 fotos que después gracias al aprendizaje de haar cascade se tomaron las características de cada billete y se arrojó un xml el cual contiene las características para después importar en Python y mediante la librería de Opencv se visualiza por computador y se identifican los billetes con su denominación.
